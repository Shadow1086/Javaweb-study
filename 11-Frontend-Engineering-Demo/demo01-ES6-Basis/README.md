# demo01 ES6 基础与模块化

这个目录按“先语法、后模块”的思路组织。

## 文件说明

- `01-letAndConst/test.html`
  - `let`、`const`、模板字符串
- `02-解构表达式.html`
  - 数组解构、对象解构、函数参数解构
- `03-拷贝问题.html`
  - 浅拷贝、深拷贝、`spread` 的常见用法
- `04-分别导出`
  - 先定义，再统一 `export`
- `05-同一导出`
  - 在声明时直接导出
- `06-默认导出`
  - `default export` 与普通导出混用

## 学习提示

1. ES6 模块化示例需要通过浏览器打开 `index.html`。
2. `script` 标签要写 `type="module"`，否则 `import` / `export` 会报错。
3. 浏览器环境里，导入本地模块时要写完整文件名。

## 这一阶段先别急着学什么

- 先别把 npm、打包工具、脚手架混进来
- 先把“模块是怎么拆开的”这件事弄明白
- 否则后面看到 Vite、Webpack 时，会只会敲命令，不知道它们在帮你做什么
